# 进制转换

## 进制转换函数: conv

**语法**: conv(BIGINT num, int from_base, int to_base)
**返回值**: string
**说明**: 将数值num从from_base进制转化到to_base进制

```
hive> ``select` `conv(17,10,16) ``from` `iteblog;``11``hive> ``select` `conv(17,10,2) ``from` `iteblog;``10001
```
