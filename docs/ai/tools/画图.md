# AI画图方法

## gemini-cli+mermaid

直接让 gemini 生成流程图的mermaid 代码，拿到markdown 里渲染

## draw.io mcp

1. 安装浏览器插件，[drawio-mcp-extension](https://chrome.google.com/webstore/detail/drawio-mcp-extension/okdbbjbbccdhhfaefmcmekalmmdjjide)
2. 启动mcp

```shell
cd ~/code/AI
git clone https://github.com/lgazo/drawio-mcp-server.git
npx -y drawio-mcp-server --transport http --http-port 3000
# or both: npx -y drawio-mcp-server --transport stdio,http --http-port 4000
curl http://localhost:3000/health
# { "status": "ok" }
```

此时，drawio-mcp-extension插件会用绿色提示已连接mcp

3. 打开 [draw.io](https://app.diagrams.net/)网页，会多出一个图标，点击即可对话并生成流程图
