# 快捷键

## 标签

```
新建标签：command + t
关闭标签：command + w
切换标签：command + 数字 command + 左右方向键
切换全屏：command + enter
查找：command + f
```

## 分屏

```
垂直分屏：command + d

水平分屏：command + shift + d

切换屏幕：command + option + 方向键 command + [ 或 command + ]

查看历史命令：command + ;

查看剪贴板历史：command + shift + h
```

## 其他

```
粘贴：中键
⌘ + shift + h 会列出剪切板历史
```
