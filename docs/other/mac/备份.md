# 挂载网盘

## 挂载阿里云盘

[参考](https://www.macappbox.com/a/mac-shang-gua-zai-a-li-yun-pan-wei-ben-di-ci-pan-de-jiao-cheng.html)

1. webdav-aliyundriver 将阿里云盘通过网络接口模拟成为一个webdev接口，从而通过第三方客户端挂载到本地磁盘

pip install aliyundrive-webdav

2. 使用 CloudMounter 挂载磁盘

测试结果： 挂载后，Finder 不响应了。
