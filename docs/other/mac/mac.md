# shell

```shell
查看所有shell
cat /etc/shells
/bin/bash
/bin/csh
/bin/ksh
/bin/sh
/bin/tcsh
/bin/zsh

查看当前shell
echo $SHELL

切换shell
chsh -s /bin/bash
```
