## wifi连接无法上网

[参考](https://blog.csdn.net/lyxleft/article/details/79971963)

```shell
1、打开系统偏好设置—>网络—>WiFi—>高级—>WiFi—>删除首选网络框内的所有网络—>点击好—>点击应用；

```

# 休眠命令

## 命令说明

1. 命令如下：
    `sudo shutdown [-h | -r | -s] [time]`
    **此命令需要管理员权限**
2. 参数说明：

- `-h` ：关机（halt）

- `-r` ：重启（reboot）

- `-s` ：休眠（sleep）

- ```
  time
  ```

   ：执行操作的时间

  - yymmddhhmm ：指定年月日时分，如 17022318 表示2017年2月23日18时
  - now ：现在
  - +n ：n分钟后
  - hh:mm ：今天某时某分

1. 其他关机和重启命令
    立刻关机：`sudo halt`
    立刻重启：`sudo reboot`
