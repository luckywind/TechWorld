新建映射

| 选项                                 | 描述                                                         |
| ------------------------------------ | ------------------------------------------------------------ |
| 应用名称                             | 自定义填写                                                   |
| 映射类型                             | 选择HTTPS                                                    |
| 外网[域名](https://domain.oray.com/) | 选择当前帐户下可用的加密壳域名，用作外网访问地址 （[点击开通](https://hsk.oray.com/parts-center/https.html)） |
| 外网端口                             | 默认为443                                                    |
| 内网主机                             | 服务器内网IP地址                                             |
| 内网端口                             | 内网服务端口                                                 |
| 带宽                                 | 默认根据账号等级分配相应的带宽，也可根据实际需求额外付费购买，每条映射带宽上限为10Mbps |

![image-20260514150749251](https://piggo-picture.oss-cn-hangzhou.aliyuncs.com/image-20260514150749251.png)

添加成功后，会生成一个外网访问地址；外网通过花生壳映射生成的访问地址，即可直接访问到用户所搭建的服务。

**①花生壳程序是映射本地内网服务实现外网访问**
**②若异地登录花生壳，将会导致已设置的本地内网服务映射失效**
**③若有多个地点的服务需要发布，建议通过多个账号进行映射操作**
